There are exactly 100 observations, including one blank observation (Unsung Hero, in observance of all the women who made a difference in 2020 but could not be fit on the list). Visualizing this type of data is not my strong suit. It focuses more on aesthetics than Im used to. Put another way, the data cant tell me which visualization to use. Although thats not necessarily true. But theres no network, no obvious relationships between the women, and not much data. Lets take a look at some of the variables, though, to get a sense.

A long description variable lends itself to text analysis, and the spiky area of what makes a winner a winner, a problem fraught with danger because (among other things no doubt) it lacks a counterfactual. A better visualization might break down the winners by category or role. Probably there arent enough roles in common, but there are only five unique categories.

In fact, there are only five categories, and one is All, which contains one representative: the unsung hero. Well, a nice network could show the way things are related. There are 75 roles, so only a couple repeat. Lets see.

women_graph_lvl1 <- data.frame(
  name = "BBC 100 Women",
  level = 1,
  category = NA,
  activist = NA
)

women_graph_lvl2 <- women %>% 
  select(name = category) %>% unique() %>% 
  mutate(level = 2, category = NA, activist = NA)
  
women_graph_lvl3 <- women %>% 
  filter(!grepl("activist", role, ignore.case = TRUE)) %>% 
  select(name, category) %>% 
  rbind(data.frame(name = "Activist", category = NA)) %>% 
  mutate(level = 3, activist = FALSE)

women_graph_lvl4 <- women %>% 
  filter(grepl("activist", role, ignore.case = TRUE)) %>% 
  select(name, category) %>% 
  mutate(level = 4, activist = TRUE)
  

women_graph_vertices <- rbind(women_graph_lvl1, 
                              women_graph_lvl2, 
                              women_graph_lvl3,
                              women_graph_lvl4)

women_graph_activist_lyr <- women %>% 
  filter(grepl("activist", role, ignore.case = TRUE)) %>% 
  select(to = name) %>% 
  mutate(from = "Activist") %>% 
  unique()

women_graph_df <- women %>% 
  mutate(from = "BBC 100 Women") %>% 
  select(from, to = category) %>% 
  unique() %>% 
  mutate(role = NA) %>% 
  rbind(select(women, from = category, to = name, role)) %>% 
  mutate(
    to = case_when(
      grepl("activist", role, ignore.case = TRUE) ~ "Activist",
      TRUE ~ to
    )
  ) %>%
  select(from, to) %>%
  rbind(women_graph_activist_lyr)
